# Task
提示された医学記述の「科学的正確性」「論理的因果関係」「一般性（標準ガイドラインに沿っているか）」を批判的に検証し、修正箇所の特定と代替案を提示
長い文章の場合も上から1パラグラフずつ検討し、網羅的に検討

# Criteria
1. 機序の正確性と反証可能性
   - 解剖学、生理学、病態生理学、薬理学の基本原則に違背していないか。
   - 因果関係の逆転、または論理の飛躍（例: 「AであるためB」の間に必要な中間因子Cの欠落）はないか。

2. 一般性とコンテキストの限定
   - 記述の適用範囲（対象患者の年齢、性別、基礎疾患、重症度）が適切に限定されているか。
   - 特異な症例報告の知見を、普遍的な治療方針であるかのように過度な一般化していないか。
   - 地域性（例: 欧米と日本のガイドラインの差異、保険適応の有無）が考慮されているか。

3. 記述の十分性
   - 記述は十分なものであるか。
   - びまん性肺胞障害とDICを両方引き起こす疾患について、DICのみを言及していたりしないか
   - 禁忌や例外を無視していないか

4. エビデンスの階層
   - 記述の根拠が、どの階層（メタアナリシス、RCT、コホート、専門家の意見）に位置づけられるか。

5. 解答の一意性
- 多くの研究者、医師、医学生、にとって一般的な回答であるか

# 出力フォーマット
評価的な形容詞（「素晴らしい」「不十分な」など）は使用せず、客観的事実のみを記述

1. 問題がある箇所ごとに、以下を1セットとして列挙
    - 【対象箇所】: （原文の該当テキストを引用）
    - 【不正確・不適切な理由】: （機序の誤り、過度な一般化、データの誤りなどを論理的に指摘）
    - 【修正案】: （そのまま置き換え可能な正確な文章）

2. 保留・要確認事項
