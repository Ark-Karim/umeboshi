# Qualoty criteria

## MUST

高校生でも十分に理解できるように, 難しい概念に比喩を『』で併記。理解しやすくする。高校生でも十分に理解できるようにするために、十分な数、十分な詳しさの比喩をつける。

『』を使用して比喩を明記する. 『』内に 読点 `。` を含めてはいけない. 

- 例: ユビキチン化 『分解・廃棄のタグ付け』
- 例: Cascade 『ドミノ倒し』

## SHOULD

none

## COULD

抽象化：日常的・個別具体的なエピソードは描写せず、社会構造、組織システム、または物理現象へと抽象化した簡潔な表現に落とし込む。
