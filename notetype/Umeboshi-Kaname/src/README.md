# Umeboshi Kaname Note Type

Based on AnKing Overhaul with additional features:

- LLM Question Section (Claude-compatible API)
- localStorage-based settings
- Settings modal (gear icon)

## Features Inherited from AnKing Overhaul
- Cloze deletion with One-by-One mode
- 14 hint buttons with keyboard shortcuts
- Colorful tags
- Countdown timer
- Image blur/zoom
- Wikipedia searches
- Night mode & mobile support

## New Features
- **LLM Chat**: Ask questions about card content directly from the review screen
- **Settings Gear**: Configure font size, timer, tags, LLM settings from the card
- **Template Buttons**: Pre-built medical question templates
